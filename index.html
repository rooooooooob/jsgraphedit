Controls:

  • Insert Mode:
    • Left click on a vertex to select it, and create an edge to it if there is a vertex already selected.
    • Click not on a vertex to create a new vertex there. If a vertex is selected, an edge will be created too.
  • Remove Mode:
    • Left click on a vertex to select it, and remove an edge from it if there is a vertex already selected.
    • Click on a selected verex to delete it.
  • Mode Mode:
    • Left click on a vertex to select it and left click elsewhere to move the selected vertex there.
